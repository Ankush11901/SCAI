Export Platform Redesign

UI/UX Upgrade for Integrated CMS Exports

Overview

The Export Article modal has been redesigned to provide a more intuitive, connection-aware, and visually structured interface for publishing content to external platforms.

Key Improvements

Platform Grid

Transitioned from a list view to a 2-column card grid for better scanning.

Status Indicators

Real-time "Connected" / "Not Connected" states for all supported platforms.

Enhanced Header

Simplified hierarchy with clear primary title and secondary instruction.

Action Branding

Updated all CTAs to "Export Now" for increased urgency and clarity.

Technical Components

The implementation resides primarily within:

components/cms/CMSExportDialog.tsx

The selection logic combines WordPress site connections with generic CMS platform configurations, mapping them to visual cards with specific brand background colors.

Design Constraints

  • Grayscale UI: All functional elements respect the project's grayscale restriction.
  • No Hover Reliance: Status and selection states are explicit and do not require hover interactions.
  • Consistent Spacing: Uses a 12px gap for the grid and 16px padding for cards.
