⭐ GOLDEN RULES (Mandatory)

  • Header Consistency - H1 type determines ALL H2 types
  • Color Restriction - Black/White/Grayscale ONLY
  • No Emoji - Only ✓, •, ★, ☆, +/– allowed
  • One Type Per Article - Cannot mix article types
  • H1 Exclusivity - Exactly ONE H1 per article
  • H3 Restriction - Only in FAQ and Honorable Mentions
  • AI Cannot Modify - No fabrication of external data

📏 CHARACTER LIMITS

  • H1: 30-70 characters
  • H2: 30-70 characters
  • Closing H2: 30-70 characters
  • FAQ H3: 20-70 characters
  • Meta Title: 30-60 characters
  • Meta Description: 120-160 characters
  • Featured Image Alt: 80-125 characters

📝 WORD COUNT TARGETS

  • Overview Paragraph: 100 words (2×50)
  • Standard Paragraph: 150 words (3×50)
  • Closing Paragraph: 150 words
  • FAQ Answer: 125 words each
  • All with ±5 tolerance

🔤 FORMAT CONSISTENCY

  • QUESTION: Start with What/How/Why/Which/When/Where, end with ?
  • STATEMENT: Declarative sentences, NO question marks
  • LISTICLE: Start with number (1., 2., 3., ...)
